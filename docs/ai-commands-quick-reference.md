## Keyboard Shortcuts
- `/` - Open AI command menu
- `Esc` - Close menu
- `Enter` - Execute custom prompt

## Tips
- Write at least a paragraph before using AI commands
- Use search in slash menu to find commands quickly
- Try "Custom Prompt" for unique requests
- Commands work on entire document content
- Some commands add content, others replace it

## Toolbar Buttons
- **AI Assistant** - Dropdown with quick actions
- **Custom Prompt** - Enter any AI instruction
- **Copy** - Copy markdown to clipboard
- **Download** - Save as .md file
