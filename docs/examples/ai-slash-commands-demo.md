## Tips for Best Results

1. **Start with substance**: Write at least a paragraph before using AI commands
2. **Use the right command**: Match the command to your goal
3. **Iterate**: Run multiple commands to refine content
4. **Combine commands**: Use several in sequence for best results
5. **Review output**: Always check AI-generated content for accuracy
